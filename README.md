# Cafe Diploma System

Дипломный проект: информационная система учета продуктов и анализа финансовых показателей предприятия общественного питания.

## Запуск в PowerShell

```powershell
pip install -r requirements.txt
Get-Content schema.sql | mysql -u root -p
python app.py
```

Если `mysql` не распознан, укажите полный путь к `mysql.exe`.

## Вход

Логин: `admin`  
Пароль: `admin`

## Основные разделы

- Панель управления
- Товары
- Продажи
- Расходы
- Склад
- Финансы
- Аналитика
- Отчеты
