SmartCafe — готовая версия проекта.

Запуск:
1) Откройте папку проекта в VS Code.
2) Убедитесь, что MySQL запущен.
3) Если нужно обновить названия складов и гостя, выполните в MySQL файл smartcafe_update.sql.
4) Запустите: python app.py
5) Откройте: http://127.0.0.1:5000
