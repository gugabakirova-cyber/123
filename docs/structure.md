# Структура проекта

Файл создан заранее для письменной части диплома.

- `app.py` — маршруты Flask и основная логика веб-приложения.
- `db.py` — подключение к базе данных и универсальные функции запросов.
- `config.py` — настройки проекта.
- `models/` — работа с таблицами базы данных.
- `services/` — аналитика, финансы, отчеты.
- `templates/` — HTML-страницы интерфейса.
- `static/` — стили, скрипты и изображения.
- `schema.sql` — структура базы данных.
