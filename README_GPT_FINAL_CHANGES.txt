Финальная версия SmartCafe CRM — внесенные изменения:

1. Убраны из бокового меню вкладки: Финансы, Товары, Отчеты.
2. Страница выбора склада оставлена отдельной, кнопки-стрелки сделаны визуально красивее.
3. На главной добавлены маленькие подсказки при наведении на график продаж и круговые диаграммы.
4. Во вкладке Склад добавлена кнопка Печать.
5. Кнопки PDF/Excel в Продажах, Расходах, Складе, Движении склада и Истории ревизий скачивают файлы через маршрут /download-report.
6. Импорт накладной теперь:
   - оприходует товары на склад;
   - автоматически создает расход «Закупка по накладной» на сумму накладной.
7. Импорт отчета iiko теперь:
   - добавляет позиции в Продажи;
   - списывает ингредиенты по техкартам со склада;
   - допускает уход остатка в минус, если ингредиента недостаточно.
8. В техкартах убрана колонка «Состав». Состав остался во всплывающей подсказке при наведении на «Выход порции».
9. Для ручной продажи добавлено ограничение: итоговая сумма одной продажи не больше 1 000 000 ₸.

Запуск:
1) Открыть папку cafe_diploma_system
2) pip install -r requirements.txt
3) python app.py
